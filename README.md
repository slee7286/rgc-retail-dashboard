# RGC Retail Dashboard

Skeleton repository for the Really Good Culture technical assessment.

## Structure

- `app/frontend` - React/Next.js dashboard UI
- `app/backend` - Node/Express API service
- `scripts` - data preprocessing and pipeline support
- `Dataset` - raw files used for preprocessing
- `docker-compose.yml` - local multi-container development
